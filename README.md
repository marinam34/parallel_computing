# Parallel Computing


Репозиторий содержит реализацию и сравнение нескольких параллельных вариантов решения трёхмерного уравнения волны с аналитическим решением:

\[
\frac{\partial^2 u}{\partial t^2} = a^2 \Delta u
\]

Используется явная двухслойная разностная схема второго порядка точности по пространству и времени, периодические граничные условия по \(x, y\) и нулевые по \(z\). В качестве теста используется функция с известным аналитическим решением, что позволяет контролировать погрешность.

Основная цель работы — сравнить производительность следующих вариантов:

- последовательная реализация;
- OpenMP (распараллеливание по нитям);
- MPI (распараллеливание по процессам с декомпозицией области);
- гибрид MPI+OpenMP;
- MPI+CUDA (GPU).
